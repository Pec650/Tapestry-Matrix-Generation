# Tapestry-Matrix-Generation

## Expected Output:

A matrix with 5 different letters (symbols) randomly positioned.

### Example:

|---|---|---|---|
| R |   |   | T |
|---|---|---|---|
|   |   | X |   |
|---|---|---|---|
| V |   |   |   |
|---|---|---|---|
|   |   |   | T |
|---|---|---|---|

R, X, T, and V are symbols while the empty cells are "fillers".
